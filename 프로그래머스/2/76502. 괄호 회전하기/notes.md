## ⚠️ 내가 빠진 함정

_(조건을 잘못 해석? 시간복잡도 착각?, 자료구조 잘못 선택?)_

- Java의 Pass By Value에 대해 Reference Type를 Primitive Type과 헷갈렸다. 참조타입의 주소 값을 넘겨주기 때문에 메서드에서 해당 참조타입을 파라미터로 입력 받고 변경하면 원본에도 변경이 된다.
- `회전`을 더  우아하게 처리할 수 있었다. FIFO처럼 보여서 Queue를 사용했는데, 굳이 그럴 필요 없었다! 

## 🔁 다음에 이 유형 나오면

_(트리거 문장 : 이 표현이 보이면, 먼저 __________ 를 떠올린다.)_

- '회전'이라는 말이 나오면, 리스트를 이어붙여서 인덱스를 순회할 수 있는지 생각해보자.
- double for-loop에서 내부 반복문을 멈추고 다음 외부 순회를 시작하려면 아래와 같이 사용할 수 있다.
```java
    A:for (int i = 0; i < n; i++) {
        for (int j = i; j < i + n; j++) {
            continue A;        
        }
    }
```
