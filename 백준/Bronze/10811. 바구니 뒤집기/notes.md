# BOJ 10811

## 문제 요약
배열 인덱스 지정된 범위에 대해서 역순으로 배열 요소 바꾸기

## 핵심 아이디어
역순으로 바꾸는 것을 얼마나 간결하게 코드로 표현하냐가 관건이었다.

## 시간복잡도
O(N)

## 내가 처음 사용한 접근
대상이 되는 배열과 똑같은 크기의 가진 임의의 temp 배열 만들어서 반복문으로 복사, 역순으로 할당, temp 배열 0으로 초기화

## 실수
문제 자체는 맞았지만, 인덱스가 헷갈려서 한참을 반복했다. 발상은 단순하나 코드 가독성이 좋지 않아서 발생한 일이었다.

## 해결 방법
while loop과 후위연산을 사용했다면 더 간단하게 풀 수 있었다. 인덱스를 골치아프게 생각할 필요가 없었다.

## 배운 점
첫번째 떠오른 발상이 과연 효과적인지 다시 한 번 검토해보는 것이 필요하다. 일단 초반 코드조차도 temp 배열을 굳이 매번 초기화 할 필요가 없었다.
